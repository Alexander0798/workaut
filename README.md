 # Интернет магазин воркату.рф с админ панелью доступной только администратору через вход логин и пароль, отправка ошибок админу в телеграм и на емеил в случае ошибки отправки в тг, отправка менеджеру заявок в тг и на емаил.
 ## Сайт написан с нуля в соло ~ за 2.5 месяца из требование быстрого запуска mvp на что повлияло отсутствие комментариев и тестов, на данный момент сайт запушен на vds и активен по домену воркаут.рф, но требует рефакторинг и комментариев и тестов и самое важное это перенастройка ssr, внедрения вебсокетов для запуска чата с менеджером...
 ### Стек фронта - react, ts, webpack, scss, 
 ### Стек Бэк - nestjs, ts, postgresql
 ### Server - ubuntu 22.04, nginx, pm2, cron для автоперевыпуска ssl сертификата через certbot
