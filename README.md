 # Интернет магазин Воркату.рф с админ панелью доступной только администратору через вход логин и пароль, отправка ошибо админу в телеграм и на емеил в случае ошибки отправки в тг, отправка менеджеру заявок в тг и на емаил.
 ## Сайт написан с нуля мной в соло ~ за 2 месяца из требование быстрого запуска nvp на что повлияло отсутствие комментариев и тестов, на данный момент сайт запушен на vds и активен по домену воркаут.рф, но требует рефакторинга и комметраиев и тестов и самое важное это перенастройка ssr, внедрения вебсокетов для запуска чата с менедером...
 ### Стек фронта - react, ts, webpack, scss, 
 ### Стек Бэк - nestjs, ts, postgresql
 ### Server - ubuntu 22.04, nginx, pm2, cron для автоперевыпуска ssl сертифата через certbot
